Location mode

Local climate hazard: obtained by bilinear sampling on the spatial grid ( portugal_climate_grid.json) using the same layers shown on the map. Use exact coordinates or the reference municipality centroid.

Changing the municipality centres the map on that centroid (reference). Climate hazard always comes from the spatial grid, not legacy per-municipality values.

Fills questionnaire fields only (socio-economic, housing, health, etc.). Does not change map location. Set the point separately.

Click the map to set your location. Drag the marker to adjust. Basemap: OpenStreetMap (internet required). Layers control: top right (click to expand). Legend: bottom left.
